Super Mariochisme - a difficult platformer game

Made by:
Andreas H. Eike
Pedro Alves
Endre Rødland
Joar Østefjells
Fredrik Throndsen Hove
Martin Jakimovski

Controls:
Jump - W or Up Arrow
Go Left/Right - A/D or Left Arrow/Right Arrow

Objective: get to the end of the 9 levels, overcome all the hidden traps throughout!

Installation:
Open mpain/cmake-build-debug/mpain.exe to run the program. Can also run with a C++ IDE (such as CLion), but this requires installation of the c++ libraries below:

JsonCpp
TGUI
SFML
